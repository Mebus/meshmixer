meshmixer by johnfredcee

Convert a wide variety of 3D File Formats to Ogre Meshes via the Open Asset Import Library.

Useful for importing meshes exported from other 3D packages into Ogre: allows a large variety of post processing operations on the mesh.

Dependencies

OGRE, http://www.ogre3d.org (no greater than 1.49) wxWidgets: http://www.wxwidgets.org (2.8) boost, http://www.boost.org assimp http://assimp.sourceforge.net/

Install

Edit the .vcproj or SConstruct to point to the locations of your dependencies. Build and run from the build directory.
